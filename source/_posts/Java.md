## 常用CMD命令

- 盘符名称+冒号
  - 说明: 盘符切换
  - 距离: E:回车,表示切换到E盘
- dir
  - 说明: 查看当前路径下的内容
- cd目录
  - 说明： 进入单级目录
- cd ..
  - 说明：回退到上一级文件夹

- cd 目录1\目录2\\...
  - 说明： 进入多级目录
- cd \
  - 说明： 回退到盘符目录
- cls
  - 说明：清屏
- exit
  - 说明：退出命令提示符窗口

>**环境变量**
>
>用来记录系统查找路径，实现在任意目录下都可以打开指定的软件。

## 变量

### 变量的定义格式

数据类型  变量名  =  数据值;

### 变量的使用方式

#### 输出打印

```java
int a = 10;
System.out.println(a);		//10
```

#### 参与计算

```java
int a = 10;
int b = 20;
System.out.println(a + b);  //30
```

#### 修改记录的值

```java
int a = 10;
System.out.println(a);  //10
a = 20;
System.out.println(a);	//20
```

### 注意事项

- 只能存一个值
- 变量名不允许重复定义
- 一条语句可以定义多个变量,中间用逗号隔开
- 变量在使用之前一定要进行赋值

## 标识符

### 定义

标识符:给{% label 类,方法,变量 red%}等起的名字

### 命名要求

#### 硬性要求

- 由数字,字母,下划线和美元符组成
- 不能以数字开头
- 不能是关键字
- 区分大小写

#### 软性建议

- 小驼峰命名:{% label 方法,变量 red%}
  - 第一个单词全部小写,之后单词首字母大写
- 大驼峰命名: {% label 类名 red%}
  - 每个单词首字母大写

## 键盘录入

>Java帮我们写好一个类叫Scanner,这个类就可以接收键盘输入的数字

### 步骤

- 导包

  - ```java
    import java.util.Scanner;
    ```

- 创建对象

  - ```java
    Scanner sc = new Scanner(System.in);
    //sc 是变量名,可以变,其他都不可以变
    ```

- 接收数据

  - ```java
    int i = sc.nextInt();
    // i 是变量名可以变,其他的不允许变
    ```

### 数据类型转换

#### 隐式转换(自动类型提升)

取值范围小转换为取值范围大

- 取值范围小的,和取值范围大的进行运算时,小的会自动提升为大的
- byte short char 三种类型的数据在运算的时候,都会先提升为int

#### 强制转换

取值范围大转换为取值范围小

>**取值范围**
>
>double>float>long>int>short>byte

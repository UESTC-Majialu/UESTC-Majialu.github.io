Java入门

Java是什么

人机交互

打开CMD

常用CMD命令

  • 盘符名称+冒号
    • 说明: 盘符切换
    • 距离: E:回车,表示切换到E盘
  • dir
    • 说明: 查看当前路径下的内容
  • cd目录
    • 说明： 进入单级目录

  • cd ..

    • 说明：回退到上一级文件夹

  • cd 目录1\目录2\…

    • 说明： 进入多级目录
  • cd \
    • 说明： 回退到盘符目录
  • cls
    • 说明：清屏
  • exit
    • 说明：退出命令提示符窗口

环境变量

用来记录系统查找路径，实现在任意目录下都可以打开指定的软件。

Java程序初体验

下载JDK

官网：http://www.oracle.com

安装目录

  • bin: 该路径下存放了各种工具命令
    • javac和java
  • conf：该路径下存放了相关的配置文件
  • include： 该路径下存放了一些平台特定的头文件
  • jmods: 该路径下存放了各种模块
  • legal: 该路径下存放了各模块的授权文档
  • lib: 该路径下存放了工具的一些补充JAR包

Java的前世今生

Java可以做什么

Java SE

Java语言的标准版，用于桌面应用的开发，是其他两个版本的基础

Java ME

Java语言的(小型版)，用于嵌入式电子设备或者小型移动设备。

Java EE

Java语言的(企业版)，用于Web方向的网站开发。

面向对象

安全性

多线程

简单易用

开源

跨平台

原理： 虚拟机，JVM(Java Virtual Machine)

JRE和JDK

  • JDK(Java Development kit,java开发工具包)

    • JVM
    • 核心类库
    • 开发工具
      • javac:编译工具
      • java:运行工具
      • jdb: 调试工具
      • jhat: 内存分析工具

  • JRE(Java Runtime Environment, Java运行环境)

    • JVM
    • 核心类库
    • 运行工具

Java基础概念

注释和关键字

注释

  • 单行注释： //
  • 多行注释： /* */
  • 文档注释： /** **/

关键字

关键字的特点：

  • 关键字的字母全部小写
  • 常用编辑器会对关键字高亮

字面量

字面量类型 说明 举例
整数类型 不带小数点的数字 66，-88
小数类型 带小数点的数字 13.14，-5.20
字符串类型 用双引号括起来的内容 “Hello World” “黑马”
字符类型 用单引号括起来的，内容只能有一个 ‘A’ ‘0’ ‘我’
布尔类型 布尔值，表示真假 true,false
空类型 特殊的值，空值 null
\t 制表符 打印的时候，把前面字符串的长度补齐到8，或者8的整数倍。最少补1个空格，最多补8个空格

数据类型 变量名 = 数据值;

参与计算

修改记录的值

注意事项

  • 只能存一个值
  • 变量名不允许重复定义
  • 一条语句可以定义多个变量,中间用逗号隔开
  • 变量在使用之前一定要进行赋值

数据类型

基本数据类型

数据类型 关键字
整数 byte
short
int(默认)
long:数据值的后面需要加一个L作为后缀
浮点数:数据值的后面需要加一个F作为后缀 float
double(默认)
字符 char
布尔 boolean

标识符

定义

标识符:给类,方法,变量 等起的名字

命名要求

硬性要求

  • 由数字,字母,下划线和美元符组成
  • 不能以数字开头
  • 不能是关键字
  • 区分大小写

软性建议

  • 小驼峰命名:方法,变量
    • 第一个单词全部小写,之后单词首字母大写
  • 大驼峰命名: 类名
    • 每个单词首字母大写

数据类型转换

隐式转换(自动类型提升)

取值范围小转换为取值范围大

  • 取值范围小的,和取值范围大的进行运算时,小的会自动提升为大的
  • byte short char 三种类型的数据在运算的时候,都会先提升为int
